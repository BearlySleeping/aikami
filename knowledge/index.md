# Welcome to Aikami Docs

> The unified single source of truth for Aikami architecture, conventions, and developer coordination.

## What is Aikami?

Aikami is a modern monorepo application platform built with SvelteKit, Firebase, and Bun. It provides a Progressive Web App (PWA) with real-time features, authentication, and a shared component library.

- **SvelteKit PWA** — Cross-platform progressive web app
- **Firebase Backend** — Serverless functions, Firestore, authentication
- **Bun Runtime** — Fast JavaScript/TypeScript runtime
- **Moon Monorepo** — Task orchestration and dependency management
- **Biome** — Consistent linting and formatting
