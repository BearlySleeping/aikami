## Fixes Applied (1 iteration)

### Root Cause
The test preload (`src/lib/test_preload.ts`) mocked `gameStateService` as a generic Proxy stub. When the C-233 world-gen context injection accessed `gameStateService.worldGenOutput.npcs.length`, the stub returned a mock function for `worldGenOutput`, then `worldGen.npcs` was `undefined`, causing a `TypeError`.

### Changes
1. **`src/lib/test_preload.ts`** — Added a proper `worldGenOutput` property to the `gameStateService` mock stub, returning a default `WorldGenOutput` shape with empty arrays.
2. **`src/lib/views/combat/combat_view_model.svelte.ts`** — Added `Array.isArray(worldGen.npcs)` guard to the world context injection check for defensive safety.
3. **`src/lib/views/game/ui/overlays/dialogue/dialogue_overlay_view_model.svelte.ts`** — Same defensive `Array.isArray` guard.
