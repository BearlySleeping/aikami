# Pipeline Sound Effects

Place `.wav` files here. The pipeline plays them at key events.

| File | When |
|---|---|
| `pipeline-blocked.wav` | Pipeline reaches `blocked` state |
| `pipeline-complete.wav` | Pipeline reaches `merged` or `pr_created` |
| `pipeline-needs-input.wav` | Pipeline is waiting for human review input |

Currently these are empty placeholder files. Replace them with real sounds.
