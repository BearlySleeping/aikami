# Technology Stack

This document outlines the technology stack for the AiKami project.

- **Monorepo Tools:** Moon, Bun
- **Frontend Frameworks/Libraries:**
  - SvelteKit (for the PWA)
  - Astro (for static sites like landing page and documentation)
  - GodotJS (a fork of Godot that allows TypeScript for the game client)
- **Backend Technologies:**
  - Firebase (Functions, Authentication, Firestore, Storage)
- **AI Frameworks & Tools:**
  - Google Genkit (for production AI services)
  - Ollama & ComfyUI (for local development and advanced users)
- **Languages:**
  - TypeScript (used across the entire stack, including the GodotJS client)
- **Schema & Validation:**
  - Zod (for sharing and validating data structures between the PWA, backend, and game client)
